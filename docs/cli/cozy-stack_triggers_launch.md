## cozy-stack triggers launch

Creates a job from a specific trigger

```
cozy-stack triggers launch [triggerId] [flags]
```

### Examples

```
$ cozy-stack triggers launch --domain cozy.localhost:8080 748f42b65aca8c99ec2492eb660d1891
```
