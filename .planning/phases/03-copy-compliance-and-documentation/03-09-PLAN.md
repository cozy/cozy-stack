
Supported methods for the documentation's method table:
| Method | Purpose | Key behavior |
|--------|---------|--------------|
| OPTIONS | Discovery | Returns `DAV: 1`, `Allow:` header; no auth required |
| PROPFIND | List/stat | Depth 0 and 1 only — `infinity` blocked with 403 |
| GET | Read file | Streams via vfs.ServeFileContent; supports Range/If-Modified-Since |
| HEAD | Stat file | Same as GET minus body |
| PUT | Upload | Streaming; supports If-Match / If-None-Match; parent must exist (409 otherwise) |
| DELETE | Trash | Soft-delete to `.cozy_trash` (not permanent destroy) |
| MKCOL | Create dir | Single level only; 405 if exists; 409 if parent missing; 415 if body |
| MOVE | Rename/reparent | Overwrite:T default; 412 on Overwrite:F + existing dest |
| COPY | Duplicate | File: `vfs.CopyFile` (or `note.CopyFile` for Cozy Notes); Dir: recursive walk; 207 on partial failure |

    ### `## Compatibility notes & limitations`
    Bullet list covering:
    - No LOCK/UNLOCK support (Class 1 only — RFC 4918 §6). macOS Finder mounts are READ-ONLY as a consequence
      (Finder requires LOCK for writes).
    - No dead properties / PROPPATCH (deferred to v2 — `ADV-V2-02` in REQUIREMENTS.md)
    - `PROPFIND Depth: infinity` rejected with 403 Forbidden (DoS protection)
    - `PROPFIND Depth: 1` supported without pagination cap — very large directories may take time
    - `.cozy_trash` directory is exposed read-only (DELETE=trash semantics; the trash itself cannot be
      written to via WebDAV)
    - Soft-delete: DELETE moves files to `.cozy_trash` instead of destroying them — use the files API
      to permanently remove
    - Streaming: PUT and GET both stream — no request/response is fully buffered
    - ETag source: content MD5 (not CouchDB `_rev` — revisions change on metadata edits)
    - Date format: RFC 1123 / HTTP-date (not RFC 3339 — macOS Finder silently misparses ISO 8601)

    ### `## Troubleshooting`
    Common errors and their causes:
    - 401 Unauthorized with `WWW-Authenticate: Basic realm="Cozy"`: token missing or expired
    - 403 Forbidden on PROPFIND with Depth:infinity: by design (§ Compatibility notes)
    - 403 Forbidden on writes to `.cozy_trash/*`: trash is read-only via WebDAV
    - 409 Conflict on PUT or MKCOL: parent directory does not exist
    - 412 Precondition Failed: `If-Match`/`If-None-Match` ETag mismatch, or Overwrite:F on existing dest
    - 415 Unsupported Media Type on MKCOL: request body present — MKCOL with body is not supported
    - 502 Bad Gateway on MOVE/COPY: `Destination` header points to a different host

    Where to find logs: reference the cozy-stack logging documentation.
