From web/webdav/move.go (the structural twin — STUDY THIS BEFORE CODING):
The move handler flow is:
1. davPathToVFSPath(c.Param("*")) -> srcPath  (403 on ErrPathTraversal)
2. parseDestination(c.Request()) -> dstPath  (400 missing / 403 traversal / 502 cross-server)
3. isInTrash(dstPath) -> 403
4. fs.DirOrFileByPath(srcPath) -> srcDir, srcFile (404 on ErrNotExist)
5. overwrite := r.Header.Get("Overwrite") != "F"
6. fs.DirOrFileByPath(dstPath) -> dstDir, dstFile (ignore ErrNotExist)
7. If exists: !overwrite -> 412; overwrite -> TrashFile/TrashDir; destExisted = true
8. fs.DirByPath(path.Dir(dstPath)) -> dstParent (409 on ErrNotExist)
9. VFS verb (ModifyFileMetadata/ModifyDirMetadata) — for COPY this is the fork point
10. 204 if destExisted else 201
