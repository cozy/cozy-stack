<context>
The proxy at `/tmp/cozy-webdav-proxy.go` is needed because cozy-stack is multi-tenant (routes by Host header).
It listens on :8888, rewrites Host to `brice.localhost:8080`, forwards to :8080.
The stack binary must be rebuilt from this branch (`go build -o /tmp/cozy-stack-webdav .`) because
the system-installed cozy-stack doesn't have WebDAV code.

Setup steps after PC restart:
1. `sudo apt install litmus`
2. `cd /home/ben/Dev-local/cozy-stack-feat-webdav`
3. `go build -o /tmp/cozy-stack-webdav .`
4. `/tmp/cozy-stack-webdav serve --disable-csp &`
5. `go run /tmp/cozy-webdav-proxy.go -instance brice.localhost:8080 &`
6. `TOKEN=$(/tmp/cozy-stack-webdav instances token-cli brice.localhost:8080 io.cozy.files)`
7. `litmus http://localhost:8888/remote.php/webdav/ onlyoffice $TOKEN`
</context>
